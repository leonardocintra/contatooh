# Contatooh

## O projeto
```
git clone https://github.com/leonardocintra/contatooh
cd contatooh
```

## Instalacao

Instale as dependencias

- NPM
- NodeJs

```
npm install
bower install
```

Execute o projeto com `nodemon`
```
npm install nodemon -g
nodemon server.js
```
